# GeoBot – Soil Metal Detector

GeoBot is a smart system designed to detect and analyze metal content in soil using sensors and data processing techniques.

## Key Modules
- combine_soil_metals.py – Main script for soil metal data processing
- clean_global_metals.py – Data cleaning and preprocessing
- metal_model.py – ML model for metal detection and analysis

## Dataset
Due to size limitations, full datasets are not included in this repository.
Sample data and dataset references are mentioned for reproducibility.
